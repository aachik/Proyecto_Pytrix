Proyecto_Pytrix
===============

Proyecto para trabajar con algebra lineal. 

Visión del proyecto:
Se espera que este programa pueda ser usado como alternativa a matlab y otros programas de la misma área, para estudiantes de las materias de álgebra lineal y derivados, al ser un proyecto hecho por un alumno resultara mas sencillo utilizar este programa y entenderlo mejor que a otros mas profesionales hechos para fines científicos y de investigación mas avanzada.

Requisitos:
* Python 3 
* Sympy
* Tkinter
